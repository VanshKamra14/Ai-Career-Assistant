# AI Career Assistant

A full-stack AI-powered job preparation platform that analyzes resumes and generates interview preparation insights.

## Tech Stack
React.js  
Node.js  
Express.js  
MongoDB  
Gemini AI API  
JWT Authentication  
Puppeteer  

## Features
- Resume analysis using AI
- AI-generated interview questions
- Skill gap detection
- ATS optimized resume generation
- Secure authentication

## Installation

Frontend
npm install
npm run dev

Backend
npm install
npm start
